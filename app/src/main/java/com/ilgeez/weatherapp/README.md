# Weather App (Demo)

Демонстрирует реализацию приложения с информацией о погоде.

![Preview](preview.png)

Проект написан на **Kotlin** с применением **Clean Architecture**.

*   Jetpack Compose.
*   Single Activity, Navigation Compose.
*   Dagger Hilt.
*   Kotlin Coroutines.
*   Retrofit 2, OkHttp.
